IDEA永久激活

准备工作及注意事项

  1. 注意：本教程补丁来源于 https://zhile.io ，如有侵权请联系作者删除！
  2. 本项目只做学习研究之用，不得用于商业用途！
  3. 建议使用 IntelliJ IDEA Ultimate 2020.1.3 及以下版本
  4. 下载 IDEA 2020.1.3
  5. 下载破解补丁，链接: https://pan.baidu.com/s/142aTZlVgcn8FAfdLYXaEXg 提取码: npi2

激活过程

  1. 启动 IDEA ，根据自己情况选择，建议选择第二项
  2. 设置 UI 主题，建议跳过设置
  3. 选择 Evaluate for free 和 Evaluate
  4. 将 jetbrains-agent-latest.zip 拖进 IDEA 窗口，选择 Restart 重新启动
  5. 激活方式选择 License server ，然后选择 为IDEA安装
  6. 出现如下图表示激活成功，点击 是(Y) 重新启动

查看状态

  • 随意创建一个项目，然后点击 help->About
  • 显示状态 Licensed to 10214 ，其中 10241 是用户名
