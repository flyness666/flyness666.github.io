hexo博客使用LaTeX

问题说明

  • hexo 默认的渲染引擎是 marked ，但是 marked 不支持 mathjax ，所以在 hexo 中输入 LaTeX 公式无法显示。
  • 需要更换 hexo 的 markdown 渲染引擎为 hexo-renderer-kramed 引擎，更换之后即可显示 LaTeX 公式。

PyCharm永久激活

准备工作及注意事项

  1. 注意：本教程补丁来源于 https://zhile.io ，如有侵权请联系作者删除！
  2. 本项目只做学习研究之用，不得用于商业用途！
  3. 适用版本 PyCharm Professional Edition 2019.3.3 及以下版本，建议使用此版本
  4. 下载 PyCharm 2019.3.3
  5. 下载破解补丁，链接: https://pan.baidu.com/s/1BLA94pT0VYmlToZyHfMnBA 提取码: hm81
  6. 进入到 PyCharm 安装目录的 \lib 目录下
  7. 将补丁 jetbrains-agent.jar 放到 \lib 里面
  8. 出现任何问题请排查 hosts 文件是否修改，然后进行重新安装，或者删除 C:\Users\10241\.PyCharm2019.3 这个隐藏文件再重新激活
