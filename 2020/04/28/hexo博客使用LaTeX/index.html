hexo博客使用LaTeX

问题说明

  • hexo 默认的渲染引擎是 marked ，但是 marked 不支持 mathjax ，所以在 hexo 中输入 LaTeX 公式无法显示。
  • 需要更换 hexo 的 markdown 渲染引擎为 hexo-renderer-kramed 引擎，更换之后即可显示 LaTeX 公式。

安装过程

安装Kramed

  • 使用 Git Bash 输入命令

更改文件配置

  • 进入 /node_modules/hexo-renderer-kramed/lib/renderer.js ，进行修改

安装mathjax包

  • 卸载 hexo-math
  • 安装 hexo-renderer-mathjax

更新Mathjax的配置文件

  • 进入 /node_modules/hexo-renderer-mathjax/mathjax.html ，进行修改

更改默认转义规则

  • LaTeX 与 markdown 语法有语义冲突， hexo 默认的转义规则会将一些字符进行转义
  • 对默认的规则进行修改防止一些字符被转义
  • 进入 /node_modules\kramed\lib\rules\inline.js ，进行第 11 行和第 20 行的修改

开启mathjax

  • 进入 /themes 下使用的主题文件夹内，修改配置文件 _config.yml
  • 修改 mathjax 配置为 true
  • 写博客文件是也需要开启 mathjax
