Python语法笔记

基本语法

变量

  1. 一般使用下划线进行变量命名
  2. 尽量不要使用中文和拼音命名变量，也不要取过长的命名
  3. python中没有常量，一般将变量名全部大写作为常量标识

注释

单行注释用 # ，多行注释用 """"""

读取输入信息

使用 input() 读入输入信息，可在括号内注明显示的内容，但该方法接收的只是字符串，如果输入的是数字，需要用 int() 进行变量转换

格式化

直接用 % 进行格式化，或者用 format() 进行格式化

运算符

  1. + - * / %与C++用法相同
  2. python中有幂运算 ** 和取整除运算 //
  3. == != > < >= <= += -= 等赋值运算与C++相同
  4. python中没有 ++ 运算，需要写成 += 1
  5. python中逻辑运算为 and or not
  6. python中存在身份运算符 is 和 is not
  7. 三元运算符写法 a = true if complare else false

while循环

while 后接一个 else ，当循环正常结束时运行 else 中的代码

列表

  1. 追加，使用 append() 将数据追加到尾部
  2. 插入，使用 insert() 将数据插入到任何位置
  3. 合并，使用 extend() 将该列表与另一列表合并
  4. 可在列表中嵌套列表
  5. del 删除，使用 del 删除列表或列表某个元素
  6. pop 删除，使用 pop() 删除最后一个或指定元素并返回该元素
  7. 清空，使用 clear() 清空该列表
  8. 修改，利用下表进行列表修改，可以使用正向下标或逆向下标
  9. 查找，使用 index() 返回从左开始匹配到的第一个元素的下标
  10. 计数，使用 count() 返回元素的个数
  11. 切片，使用 name[start:end] 取出列表中多个值，选取范围前闭后开
  12. 步长，允许跳着取值，默认步长为 1 ，步长可取负值
  13. 排序，使用 sort() 进行排序，根据 ascll 表进行的排序
  14. 反转，使用 reverse() 进行列表反转
  15. 循环，使用 for 进行列表循环
  16. 长度，使用 len() 返回列表的元素个数

元组

  1. 元组与列表相似，只是将 [] 变为了 ()
  2. 元组内的元素不可变，又被称为只读列表
  3. 元组内的 list 等可变元素中的元素可以改变
  4. 操作与列表相同，但元组没有修改操作

字符串

  1. 使用 ' or " or ''' 都可以表示字符串，其中多行字符串必须使用 '''
  2. 字符串能且只能跟字符串进行拼接和重复
  3. 字符串可以进行切片操作
  4. 字符串不可变，所有修改操作都相当于生成一个新的 str 变量
  5. 可以在引号前加 r 防止变量内字符被转义

字符串的一些常用操作方法：

  1. 首字母大写 capitalize() ，返回的是 str
  2. 全变小写 casefold() ，返回的是 str ，适用于所有语言
  3. 两侧填充字符 center() ，返回的是 str
  4. 计数 count() ，返回的是 int
  5. 判断结尾 endswith() ，返回的是 bool
  6. 判断开头 startswith() ，返回的是 bool
  7. 查找 find() ，返回的是 int ，下标值
  8. 格式化 format() ，返回的是 str
  9. 查找下标 index() ，返回的是 int ，下标值
  10. 是否数字 isdigit() ，返回的是 bool
  11. 是否小写 islower() ，返回的是 bool
  12. 是否空格 isspace() ，返回的是 bool
  13. 是否大写 isupper() ，返回的是 bool
  14. 全变小写 lower() ，返回的是 str ，适用于中英文
  15. 全变大写 upper() ，返回的是 str
  16. 大小写转换 swapcase() ，返回的是 str
  17. 左端对齐 ljust() ，返回的是 str ，返回一个原字符串左对齐并使用填充字符填充至指定长度的新字符串
  18. 右端对齐 rjust() ，返回的是 str ，返回一个源字符串右对齐并使用填充字符填充至指定长度的新字符串
  19. 连接字符串 join() ，返回的是 str ，将序列中的元素以指定的字符链接生成一个新的字符串
  20. 截取左端特定字符 lstrip() ，返回的是 str ，默认截取空格
  21. 截取右端特定字符 rstrip() ，返回的是 str ，默认截取空格
  22. 截取两端特定字符 strip() ，返回的是 str ，默认截取空格
  23. 替换字符 replace() ，返回的是 str ，可指定替换数量，默认全部替换
  24. 切割字符串 split() ，返回的是 list ，可指定切割字符和切割次数，默认以空格全部切割
  25. 右端切割字符 rsplit() ，返回的是 list ，从右端开始切割，可指定切割字符和切割次数，默认以空格全部切割

字典

  1. key-value 结构
  2. key 必须唯一
  3. 可存放任意多个 value 、可修改、可以不唯一
  4. 无序，查询速度快

集合

  1. 集合内的元素不可变
  2. 集合内的元素自动去重

函数

深浅copy

参数

  1. 位置参数：位置参数放在最前面，函数运行时以位置参数的顺序进行调用，位置参数必须按照函数声明的参数顺序传入
  2. 关键参数：关键参数必须放在位置参数后面，且不能与位置参数重复，多个关键参数的位置可以无序
  3. 默认参数：默认参数必须放在定义函数的最后面，在使用函数时可以不传入该参数
  4. 非固定参数：使用 *args 和 **kwargs 可以传入多个参数，其中 *args 以元组的形式显示， **kwargs 以字典的形式显示，未传入参数时会输出空元组和空字典，使用非固定参数时不可使用关键参数且需要传入默认参数

匿名函数

匿名函数即不需要指定函数名的函数，使用 lambda 定义匿名函数，匿名函数一般搭配其他函数使用，如 map() 函数

闭包和装饰器

  1. 闭包：函数定义和函数表达式位于另一个函数体内，返回的函数对象，不仅仅是一个函数对象，在该函数外还包裹了一层作用域，这使得该函数无论在何处调用，优先使用自己外层包裹的作用域
  2. 装饰器：在不改变原函数代码的情况下先运行目标函数，如添加登录验证函数，可使用装饰器使运行原函数前先运行登录验证函数

生成器和yield

  1. 生成器 generator 可以生成一个列表，但需要 next() 函数才能读出值
  2. yield 从函数中返还值并暂停该函数，需要 func.__next__() 才能继续执行

面向对象

类和继承

  1. self 表示自身， super 表示父类
  2. __init__() 为初始化函数
  3. class self(super) 括号内为继承的父类，可继承多个类
  4. __name 带有双下划线的变量为私有变量，带有双下划线的方法为私有方法

抽象继承

使用 abc 模块定义抽象类和抽象类方法

装饰器

  1. @classmethod 表示类方法，即该方法只能访问类变量，不能访问实例变量
  2. @staticmethod 表示静态方法，静态方法不能访问实例变量和类变量
  3. @property 表示属性方法，即将一个方法变为类的静态属性
